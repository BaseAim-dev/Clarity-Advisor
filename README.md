# Clarity Advisor — Landing Page

Plain HTML/CSS/JS landing page template for the Clarity Advisor brand.

## Open locally

Install the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) VS Code extension, right-click `index.html`, and choose **Open with Live Server**.

Alternatively, open `index.html` directly in your browser (note: some relative-path features work best over a local server).
